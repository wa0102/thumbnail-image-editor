# Thumbnail Image Editor

ASP.NET Core MVC と Python を組み合わせて開発した
サムネイル画像編集Webアプリです。

画像をアップロードするとAIで背景を透過し、
そのままブラウザ上でサムネイル画像を作成できます。

---

## 使用技術

- ASP.NET Core MVC
- C#
- Python
- JavaScript
- HTML
- CSS
- MediaPipe
- rembg
- Pillow
- NumPy

---

## 主な機能

- AI背景透過
- 人物画像ドラッグ
- 拡大・縮小
- 左右反転
- 背景画像設定
- 背景色変更
- テキスト追加
- フォント変更
- 縁取り
- 合成画像保存

---

## 開発環境

- Windows 11
- Visual Studio 2022
- Python 3.x

---

## 起動方法

### Pythonライブラリ

```bash
pip install -r requirements.txt
```

### ASP.NET Core起動

```bash
dotnet run
```

ブラウザで

```
https://localhost:7037
```

へアクセスしてください。
